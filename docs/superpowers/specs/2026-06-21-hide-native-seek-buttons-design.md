# 隐藏播放器原版前进/后退按钮

## 目标

隐藏播放器控制栏中带圆形箭头的原版“后退 10 秒”和“前进 10 秒”按钮，保留脚本新增的 `−10`、`+10` 按钮以及键盘方向键快退/快进功能。

## 方案

在脚本注入的播放器样式中增加一条限定于 `.video-js .vjs-control-bar` 的隐藏规则，目标为原版 `.vjs-seek-button`，并显式排除带有 `data-ikanbot-seek` 标记的脚本按钮。使用 CSS 隐藏而不修改 Video.js 内部状态，因此播放器控制栏被动态重建时规则仍然有效。

不删除 DOM 节点，不改变 `seekBy()`、自定义按钮创建逻辑、按钮顺序、键盘事件或进度保存逻辑。

## 安全边界

- 选择器仅作用于 Video.js 控制栏中的原版跳转按钮。
- 脚本按钮以 `data-ikanbot-seek="-10"` 和 `data-ikanbot-seek="10"` 作为保护标记。
- 其他播放、音量、进度条、倍速、画中画和全屏控件不受影响。
- 不删除任何项目文件。

## 验证

- 单元测试验证注入样式包含原版按钮隐藏规则及自定义按钮排除条件。
- 现有测试继续验证自定义 `−10/+10` 按钮只创建一次且顺序不变。
- 运行完整测试与 JavaScript 语法检查。
